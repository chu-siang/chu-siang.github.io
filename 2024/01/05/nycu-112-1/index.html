NYCU 112-1 交大修課心得

Johnson Tseng 發布於 2024-01-05

最后更新於 2024-07-08

  • 🏷️ nycu

背景：交大資財
總GPA: 4.3

必修

程式設計

Lecturer: 黃思皓

課程主要教C++  剩最後一兩周 期末大project 會使用Python 
上課方式是講簡單c++語法 然後作業主要是助教從OJ出題 回家練
期中期末都是上機 裡面的題目通常都是UVA 1~2星題 期末有一題防破台是3星的
整體對初學者還算友善
作業用的OJ題庫大概有100題左右 實際作業量大概60 但想說剩下的題也不多所以就順便刷完了 
我在高中的時候就有寫過c++ 
所以期中期末對我來說都蠻ez的

這堂課最累的應該就是期末的final projectㄌ
要寫一個量化交易的backtesting
助教有先做簡單的 data cleaning了
剩下就是看個人造化自學用TA-Lib套件去得到一些技術指標後
去找策略 對外匯市場(EUR/USD)的歷史data做backtesting 找出最好的sharpe ratio 跟 Return 

最後期中期末跟final project 都拿滿 所以也順利地拿到A+了

Project

統計學(一)

Lecturer: 巫佳煌

老師有yt影片可以輔助學習
推推巫老師的統計學 他的內容把整本的統計學都涵蓋了 
老師帶了蠻多有趣例子 讓我們比較好吸收統計學的知識
有2次期中 1次期末 期中有大會考 是共同出題 會比較難
三次考試佔 80% 剩下就是作業/小考 分數 但這學期好像比較趕
就都是作業 沒有小考 所以作業好好寫 把分數好好拿也是蠻重要的(佔比不輕QQ)
我自己是把原文書題目 還有仿間的補習班整理好題庫 都刷了一輪
配老師的影片 基本上就把考試範圍內的知識 都了解了大概

這堂課也有配置助教課 我覺得助教課也很推 會有大助教帶一些上課的重點
然後再配一些題目給我們重點練習 我都會把上課聽不懂得去找助教解
助教人也超好  只是有時候人比較多就要等一下

有些原文書比較特殊解法的就丟YT 網路上都有大神教
有時候老師會點名加分 但我的期中跟期末都90up 所以基本就A+
但老師蠻佛的 最後還會救一些人 老師講話也蠻好笑的
整體超推的一堂課

會計學(一)

Lecturer: 蔡璧徽

蔡老師也有yt影片可以輔助學習
這堂課整體蠻涼的 作業Loading不會到很重
但就是考試的佔比很重 需要讀的份量有點重 
期中也會有大會考 有一些會出考古 剩下就隨機出題
觀念跟一些基本題型記好怎麼解拿基本分問題不大
剩下的題目都是一些變化題 需要把細節分析後
才會得知原題意才解的出來的 
考前需要蠻多時間去刷考古題跟複習講義內容 
但老師給分很甜推推 老師也都會把考古題的題解影片傳連結給我們
第一堂課通常都會有爆多人加簽 如果沒超過系統上限 通常老師都會加簽
老師也教得很好 推爛

計算機概論

Lecturer: 林春成

老師的課程範圍就是把整本overview 
配分就是期中期末各占40% 作業佔20%
考試就把原文書內容讀熟 跟老師的ppt內容搞懂可以拿到一些基本分
但剩下的題目就比較不容易了
基本上都是計算跟證明題 
期中比較簡單 都是一些 位元運算 資料結構簡單實作 還有CPU的運作流程
跟寫一些machine code 實作一些加減乘除的過程

期末相比期中就比較麻煩了 證明一些code的時間複雜度 寫sql 還有data mining的一些概念手寫
分群分類之類的(這部分蠻像統計學期末的題目)
這堂課可以學到蠻多東西 loading就相對就重一點 
雖然不怎麼甜 但付出時間最後成績會反映在努力上 
老師也很認真教學 所以還是推推這堂課

其他

體育—健康體適能(甲)

Lecturer: 林怡君

這體育課很讚 我平常有健身習慣 但都沒有去拉筋 所以身體的筋都蠻硬的
這堂課老師會教我們一些拉筋的動作 還有一些肌力肌耐力的動作
比較酷的部分是期中是分組要做類似TABATA之類的 然後動作自己組內訂
算是蠻有趣的一堂課

服務學習,導師時間

免費學分課 讚啦OWOb
服學是去教會 教那邊的小孩課業 我覺得對我來說算是一個蠻好的經驗
就會構思有些複雜問題 該如何簡化 用最簡單的思維去解決困難的問題

導師時間就聽聽演講 然後寫心得 就有免費學分了超讚owob

回顧 & 期望

剛來到交大 對環境還沒有那麼熟悉 
就是一直在學習課業 也沒有參加社團 但感覺光顧好課業就佔滿了大部分的時間了
希望下學期可以繼續保持這樣的學習狀態跟成績 
然後要求自己 每天去健身運動 把狀態練好 可以在明年有更好的體態
